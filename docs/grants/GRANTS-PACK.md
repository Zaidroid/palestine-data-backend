## 2. Digital Public Goods registration (recognition; strengthens every grant)

Apply at digitalpublicgoods.net/submission. Answers to the DPG Standard:
- **SDG relevance:** SDG 16 (peace, justice, strong institutions — access to
  information), SDG 3/11 indirectly (safety of movement).
- **Open licensing:** code open source (GitHub Zaidroid/palestine-data-backend);
  data per-source licenses published at /api/v1/meta/licenses + docs/LICENSES.md.
- **Documentation:** QUICKSTART-databank.md, QUICKSTART-alerts.md, API docs live.
- **Data extraction:** full API + static dataset exports.
- **Privacy:** no user accounts on public tier, no tracking, no PII collection;
  named-fatalities data comes from published human-rights documentation.
- **Do-no-harm:** source-trust policy (docs/DATA_SOURCES.md), false-positive
  control on alerts, editorial independence clause (MONETIZATION.md §6).

## 3. Open Source Collective — fiscal hosting (the entity solution)

Apply at oscollective.org. Project: palestine-data-backend (open source, live
public API). **Before relying on it: email support@oscollective.org and ask
explicitly whether expense payouts to a Palestinian bank (Bank of Palestine /
Arab Bank, USD SWIFT) are supported** — their payout stack has a SWIFT fallback
but PS is unverified. If yes: this becomes the receiving entity for OTF/NLnet
and even org subscriptions (they invoice on your behalf).

## 4. NLnet Commons Fund (€5k–50k — prepare now, submit when call reopens after summer 2026)

Frame the "European dimension" honestly: EU-based journalists, researchers, and
humanitarian analysts are primary users of verified Palestine data; the project
strengthens the European information space on a topic saturated with
disinformation. Reuse the OTF note's problem/activities sections; NLnet loves
concrete technical milestones and open source. Their form asks for: project
name, requested amount, comparison with existing efforts (ACLED/Liveuamap —
neither is Palestine-focused, open, and provenance-first), and how funds are
spent (itemize from the OTF budget sketch).
