Quick start

Two services, both public, both CORS: *. No auth needed for read endpoints.

# 1. Latest live alerts (last hour)
curl https://wb-alerts.zaidlab.xyz/alerts/latest?limit=10

# 2. Cumulative Gaza casualties (Tech4Palestine + Gaza MoH)
curl https://api.zaidlab.xyz/api/v1/gaza/summary

# 3. ACLED conflict events for an admin2 region
curl 'https://api.zaidlab.xyz/api/v1/conflict/political-violence?admin2=Hebron&since=2026-01'

# 4. All currently-closed checkpoints
curl https://wb-alerts.zaidlab.xyz/checkpoints/closed

Live alerts

Real-time security alerts ingested from Telegram (7 channels) + RSS (3 feeds), classified into 18 event types. Every alert includes trust_score (composite confidence × source reliability), historical_boost (Insecurity Insight + ACLED corroboration), and OCHA admin1/admin2 stamps.

Checkpoints (West Bank)

781 curated + crowd-reported checkpoints. Updates flow from @ahwalaltreq Telegram channel (admin + crowd) at ~960 updates/day. Whitelist-gated to drop status-text garbage.

Client snippets

JavaScript / TypeScript

// Fetch latest alerts
const res = await fetch('https://wb-alerts.zaidlab.xyz/alerts/latest?limit=20');
const alerts = await res.json();

// Live map subscription
const ws = new WebSocket('wss://wb-alerts.zaidlab.xyz/ws');
ws.onmessage = (e) => {
  const msg = JSON.parse(e.data);
  const alert = msg.event === 'correction' ? msg.alert : msg;
  // alert.latitude, alert.longitude, alert.trust_score, alert.severity
  render(alert);
};

// Route safety
const safety = await fetch('https://wb-alerts.zaidlab.xyz/route/check', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    waypoints: [[31.7683, 35.2137], [31.5326, 35.0998]],  // Jerusalem → Bethlehem
    buffer_km: 3, since_hours: 6,
  }),
}).then(r => r.json());
console.log(safety.advisory);  // "OK" | "CAUTION" | "AVOID"

Python

import requests, websocket, json

# Cumulative Gaza summary
r = requests.get('https://api.zaidlab.xyz/api/v1/gaza/summary').json()

# All ACLED political-violence events for Hebron 2026
events = requests.get(
    'https://api.zaidlab.xyz/api/v1/conflict/political-violence',
    params={'admin2': 'Hebron', 'since': '2026-01', 'limit': 1000}
).json()

# Real-time alert stream
def on_message(ws, message):
    msg = json.loads(message)
    alert = msg['alert'] if msg.get('event') == 'correction' else msg
    print(alert['type'], alert['area'], alert['confidence'])

ws = websocket.WebSocketApp('wss://wb-alerts.zaidlab.xyz/ws', on_message=on_message)
ws.run_forever()
